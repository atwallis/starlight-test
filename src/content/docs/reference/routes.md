---
title: Endpoint Routes
---

-   **Route type** – The category of data a route returns: ONE (single object), ARRAY (list), or PAGED (paginated).
-   **One type** – A route that returns a single object from the database.
-   **Array type** – A route that returns a list of objects from the database.
-   **Paged type** – A route that returns paginated results with a total count.
-   **Custom one** – A custom route returning a single object via your handler code.
-   **Custom array** – A custom route returning a list via your handler code.
-   **Custom paged** – A custom route returning paginated data via your handler code.
