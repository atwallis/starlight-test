- **@types** - folder generated by the Restura Engine whenever there is an update
- **api.d.ts** - generated API endpoints types
- **model.d.ts**- generated Restura database table types
- **restura.d.ts** - generated types used by Restura
- **restura.schema.json** - schema file that defines API endpoints, database tables, and custom types. This is created by the restura-ui
- **restura.config.mjs** - configuration file loaded by Restura upon `init()` call
