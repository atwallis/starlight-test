---
title: Event Handlers
description: Event handlers with PostgreSQL
---

-   **Event handler** – A function that runs when a database change triggers a notification.
-   **Insert handler** – An event handler that fires when a new row is created.
-   **Update handler** – An event handler that fires when a row is modified.
-   **Delete handler** – An event handler that fires when a row is removed.
-   **Database trigger** – A PostgreSQL mechanism that notifies the app of data changes.
-   **Notify** – A table setting that enables database triggers for change tracking.
