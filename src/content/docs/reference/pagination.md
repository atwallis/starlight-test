---
title: Pagination
---

-   **Pagination** – The practice of dividing large result sets into pages for efficient loading.
-   **Page** – The current page number in a paginated request.
-   **Per page** – The number of items to return in each page of results.
-   **Total count** – The total number of matching records across all pages.
-   **Filter** – A query string in paged requests that adds dynamic where conditions.
